== Description ==

A light weight plugin that offers recent Instagram photos widget with awesome customizability options.

### Features
* Recent Instagram Photos
* Lightbox option for photos
* Custom responsive photos columns
* Photos sizes control
* Follow on Instagram button
* Light widget & Easy to use

== Installation ==

Instructions for installing the Instagram by PearlThemes Plugin.

1. In your WordPress admin go to Plugins -> Add New.
2. Enter Instagram by PearlThemes in the text box and click Search Plugins.
3. In the list of Plugins click Install Now next to the Instagram by PearlThemes.
4. Once installed click to activate.
5. Now go to Appearance > Widgets page and use Instagram by PearlThemes to your widgetized sidebars :)
